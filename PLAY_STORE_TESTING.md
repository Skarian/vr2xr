# vr2xr Play Store Closed Testing

Help us unlock public release on Google Play.

We need at least **12 testers for 14 consecutive days**.

## Join in 3 steps

1. Join the tester group  
   https://groups.google.com/g/vr2xr-testers

2. Accept the testing invite  
   https://play.google.com/apps/testing/com.vr2xr

3. Install the test app  
   https://play.google.com/store/apps/details?id=com.vr2xr

## Please keep testing active

- Install and open the app
- Keep it installed through the 14-day testing window
- Share feedback if anything breaks

Thanks for helping ship vr2xr to public release.
